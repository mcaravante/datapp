Datapp Loader

Endpoint público que sirve el bundle JavaScript que las storefronts incluyen para renderizar popups configurados desde el panel de administración.

Para incluirlo:

<script async src="https://loader.datapp.com.ar/loader.js?tenant=<TENANT_SLUG>"></script>

El bundle está en /loader.js. Esta página existe solo para que el dominio no devuelva 404 si lo abrís directamente en el browser.
